Get in touch

zsazsa is an open-source project. The fastest way to reach the project is through GitHub, but e-mail works too.

GitHub

Open an issue, start a discussion or send a pull request. This is the best place for bugs, feature requests and questions about the code.

github.com/zsazsa-project/zsazsa →

E-mail

For anything that does not belong in a public issue, reach the project by e-mail.

info@zsazsa-project.org →

A note on security reports. If you believe you have found a security issue in zsazsa, please reach out by e-mail rather than opening a public issue, so it can be handled responsibly.
